 LDT: LilDworks Development Tool
---------------------------------------------------------------------------

 LDT (LilDworks Development Tool) is a lightweight script library which can
 make the work of developers much more easier. It includes a bunch of
 collection of development scripts from project initialisation to devops.
 Enter the new level of development virtualisation, and power up your
 projects to be more portable and easier to maintain.

 For further informations about this and the releated projects,
  please visit the official site:
 
    https://www.lildworks.hu/projects/lildtools/ldt/about.html

 Have fun,
